##  
- brew: asdf, awscli, git, mas 등 개발 패키지
- cask: Docker, Chrome, VSCode 등 웹사이트에서 다운 받아 설치하는 어플리케이션
- mas: 카카오톡, Todoist 등 앱스토어를 통해 설치하는 어플리케이션 

자세한 설치리스트는 BrewFile 확인
